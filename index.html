About
Breach Data Checker — entirely local, retro UI. bring your own breach data; nothing leaves your machine. supports CSV/JSON/NDJSON/plain lists (gzip ok), IndexedDB storage, Bloom prefilter, and hashed comparisons via Web Crypto.
Usage Notes
  • File → Import Dataset… add CSV/JSON/NDJSON/plain text lists (gzip supported on modern browsers).
  • Columns understood: email, username, domain, breach, hash.
  • We store normalized emailLower, usernameLower, domain; original row is kept in raw.
  • Tools → Rebuild Index rebuilds Bloom filter and record indexes.
  • Optional Hash Compare compares your input’s SHA-1/256 to a dataset hash field.
  • View → Fit to Screen forces flow layout + page scroll if your viewport is tight.
  • Drag & drop files anywhere. Drop text to auto-check it.
Keyboard Shortcuts
